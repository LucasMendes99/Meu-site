# Portfólio Lucas Mendes

Projeto de portfólio pessoal criado a partir de um layout do Figma e desenvolvido com React, Vite e Tailwind CSS.

O projeto original está disponível no Figma:
https://www.figma.com/design/6rMSU8M6avKh0fMwTdPBvj/Portfolio-website

## Como rodar o projeto

Instale as dependências:

```bash
npm install
```

Inicie o servidor local:

```bash
npm run dev
```

Depois, abra o endereço mostrado no terminal. Normalmente será:

```text
http://127.0.0.1:5173/
```

## Tecnologias utilizadas

- React
- Vite
- Tailwind CSS
- TypeScript
- Lucide React
- Motion
