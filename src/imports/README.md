# Meu Site

Portfolio pessoal de Lucas Mendes, criado com HTML, CSS e JavaScript.

## O que tem no projeto

- Secao inicial com apresentacao
- Lista de habilidades
- Secao sobre mim
- Cards de projetos
- Formulario de contato
- Menu responsivo para celular

## Tecnologias usadas

- HTML
- CSS
- JavaScript
- Bootstrap Icons
- Google Fonts

## Como abrir no computador

Abra o arquivo `index.html` no navegador.
